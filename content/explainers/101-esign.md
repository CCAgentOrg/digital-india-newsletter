# eSign — Digital Signatures for All

## What is eSign?

eSign is an online electronic signature service in India, legally recognized under the IT Act 2000. Unlike physical digital signature certificates, eSign is Aadhaar-based and can be obtained instantly—eliminating the need to visit certification authorities.

## How eSign Works

### Process Flow
1. **Document Upload**: User uploads document needing signature
2. **Aadhaar Authentication**: Identity verification via OTP/biometric
3. **Signing**: Cryptographic signature applied to document
4. **Certificate Generation**: Signed PDF returned
5. **Verification**: Anyone can verify signature validity

### Technical Details
- **Standard**: X.509 certificate based
- **Algorithm**: RSA 2048-bit encryption
- **Certificate Validity**: Typically 1-2 years
- **Legal Standing**: Same as handwritten signature (Section 5, IT Act)

## Use Cases

### Personal
- **Property Documents**: Register property online
- **Legal Agreements**: Contracts, affidavits
- **Government Services**: Submit forms digitally

### Business
- **MCA filings**: Company incorporation, annual returns
- **GST Returns**: Signed tax submissions
- **Bank Documents**: Loan applications, agreements
- **Employment Contracts**: Remote hiring

### Government
- **Digital Locker**: Sign stored documents
- **Aadhaar Services**: Update requests
- **Income Tax**: ITR verification

## Security Features

### Authentication
- **Aadhaar OTP**: Mobile-linked number verification
- **Biometric**: Fingerprint for high-value docs
- **Consent**: Explicit permission each time

### Document Integrity
- **Tamper Evidence**: Any change invalidates signature
- **Audit Trail**: Timestamp, IP address logged
- **Certificate Chain**: Root CA verification

## Benefits

### Over Physical Signatures
- **Instant**: No waiting for courier/token
- **Paperless**: No printing/scanning
- **Cost-Effective**: ₹15-50 per signature vs ₹500+ physical
- **Accessible**: Sign from anywhere in India
- **Legally Valid**: Court-admissible evidence

## Limitations

### What eSign Cannot Do
- **Class II/III Digital Signature**: For certain legal docs requiring higher assurance
- **Some Property Registrations**: State-specific rules apply
- **Foreign Documents**: Jurisdiction limitations

### Challenges
- **Aadhaar Dependency**: Must have Aadhaar-linked mobile
- **Internet Required**: Connectivity needed
- **Document Size Limits**: Typically under 10MB

## How to eSign a Document

### Step-by-Step
1. **Choose Platform**: eMudhra, MCA, or bank portal
2. **Upload Document**: PDF format preferred
3. **Enter Aadhaar**: Linked mobile required
4. **Authenticate**: Receive and enter OTP
5. **Download Signed Doc**: Complete, legally valid
